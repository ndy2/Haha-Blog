하지만 모두 현업자 시고 현재 메시징 앱 개발을 담당하고 계시는 분도 있다고 하니 다들 잘 하시리라 생각된다. 나만 잘하면 될 것 같다.


### 3. Meet 3 @ 2023-02-15
