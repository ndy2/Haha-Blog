사놓고 어려워서 열심히 못했던 책에서 마침 선정된 프로젝트의 주제를 다룬다. 카카오톡 클론 코딩이 프로젝트의 주제이니 채팅 시스템을 다루는 것은 필수적이다.  개인적으로 채팅 메시지를 그냥 RDB에 저장하는 방식의 시스템으로 프로젝트를 진행하고 싶지는 않다. `『System Design Interview』` 의 `6장 키-값 저장소 설계` 와 `12 장 채팅 시스템 설계` 의 내용을 꼼꼼히 읽으면 좋은 설계와 팀원의 설득에 도움이 될 것 같다.
