### 1. 우선 Intellij 버전을 업데이트 해 주자.

`Preferences > Appearance & Behavior > System Settings > Updates`

> [!warning] 주의!
> 이때 꼭 check IDE for Early Access 를 Early Access Program 으로 설정해주자.

혹은 JetBrains Toolbox 를 활용해보자.
